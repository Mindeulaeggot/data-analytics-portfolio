## Featured Projects

### 1. Demand Forecasting and Inventory Optimization

Repository: [Mindeulaeggot/scm-demand-forecast-inventory-optimization](https://github.com/Mindeulaeggot/scm-demand-forecast-inventory-optimization)

Why this project matters:

- Connects predictive modeling to business decisions, not just model accuracy
- Uses retail time-series data in a supply-chain context
- Translates forecasts into safety stock and reorder point recommendations

Key highlights:

- Analyzed about `4.57 million` retail sales rows
- Compared baseline, linear regression, and random forest forecasting approaches
- Reduced MAE from `11.73` to `3.64` on the sample series
- Converted forecast output into reorder logic under lead-time and service-level assumptions

Detailed write-up: [docs/scm-demand-forecast-case-study.md](./docs/scm-demand-forecast-case-study.md)

### 2. Customer Churn Prediction

Repository: [Mindeulaeggot/customer-churn-ml-project](https://github.com/Mindeulaeggot/customer-churn-ml-project)

Why this project matters:

- Shows a complete classification workflow on business-style customer data
- Compares two ML models and evaluates them with multiple metrics
- Identifies the strongest churn-related features for retention use cases

Key highlights:

- Built a classification workflow on `64,374` customer records
- Improved test accuracy from `0.8271` to `0.9539`
- Reached decision-tree `F1 = 0.9524` and `ROC-AUC = 0.9937`

Detailed write-up: [docs/customer-churn-case-study.md](./docs/customer-churn-case-study.md)
